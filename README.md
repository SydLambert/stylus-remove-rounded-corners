# stylus-remove-rounded-corners
A Stylus userscript for removing rounded corners on Google Search. Also compatible with Stylish, although Stylus is recommended.

### Currently styled Google products
- Google Search
- Google Images
- Google News
- Google Account
- Google Support
- Google Takeout
